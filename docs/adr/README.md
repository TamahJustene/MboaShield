# Architecture Decision Records (ADR)

MboaShield 2030 uses ADRs to record significant infrastructure decisions.

## Template

Create `docs/adr/NNNN-title.md`:

```markdown
# ADR-NNNN: Title

**Status:** Proposed | Accepted | Deprecated  
**Date:** YYYY-MM-DD  
**Pillar(s):** Trust | Identity | ...

## Context

## Decision

## Consequences

## Alternatives considered
```
